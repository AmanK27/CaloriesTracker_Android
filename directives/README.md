# Directives

This directory contains Standard Operating Procedures (SOPs) and natural-language instructions (directives) for agent workflows.

## Guidelines
- Write directives in Markdown.
- Each directive should define:
  1. Goal/Objective
  2. Input data format
  3. Steps to execute (referencing specific scripts in the `execution/` folder)
  4. Output expectations
  5. Edge cases and error handling guidelines
- Update these directives as the agent learns constraints or new workflows.
